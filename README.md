# Image Partitioning Trees

PA3 for UBC's CPSC221.

Uses binary trees to represent rectangular regions of a PNG image.

## Features
- Rendering inputted tree to a PNG.
- Flipping PNG horizontally or vertically.
- Pruning of the PNG.
